`@printr/mcp` wraps the SDK as a [Model Context Protocol](https://modelcontextprotocol.io)
server, so AI agents can launch tokens, check balances, and manage wallets through typed tools.

## Run it

```bash
bunx @printr/mcp
```

Each tool ships Zod-validated input and output schemas; `structuredContent` mirrors the JSON in
