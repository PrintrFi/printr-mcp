## Prerequisites

- [Bun](https://bun.sh) (Node.js also works as a fallback runtime)
- No API key required — the SDK targets the public preview API
  (`https://api-preview.printr.money`) by default, which needs no credentials.

## Install

```bash
bun add @printr/sdk
# or run the MCP server / CLI
bunx @printr/cli setup
```

<Callout type="info">
  Full MCP-client configuration details live in
  [`INSTALL.md`](https://github.com/PrintrFi/printr-mcp/blob/main/INSTALL.md) and on the
  [MCP server](/docs/mcp) page.
</Callout>

## Your first launch

`buildToken` calls the API and returns an **unsigned** transaction payload. You then sign and
submit it (see [Wallets & signing](/docs/guides/wallets-signing)).

```ts twoslash
import { buildToken, createPrintrClient, env } from "@printr/sdk";

// Defaults target the public preview API; no key needed.
const client = createPrintrClient({
  apiKey: env.PRINTR_API_KEY,
  baseUrl: env.PRINTR_API_BASE_URL,
});

const result = await buildToken(
  {
    creator_accounts: ["eip155:8453:0xYourAddress"],
    name: "My Token",
    symbol: "MINE",
    description: "A cross-chain token.",
    chains: ["eip155:8453"], // Base
    initial_buy: { spend_usd: 10 },
    image_path: "./logo.jpg",
  },
  client,
);
```

`buildToken` returns a `neverthrow` `ResultAsync` — match on it instead of `try`/`catch`:

```ts twoslash
import { buildToken, createPrintrClient, env } from "@printr/sdk";
const client = createPrintrClient({ apiKey: env.PRINTR_API_KEY, baseUrl: env.PRINTR_API_BASE_URL });
const result = await buildToken(
  { creator_accounts: ["eip155:8453:0x"], name: "x", symbol: "X", description: "", chains: ["eip155:8453"], initial_buy: { spend_usd: 10 } },
  client,
);
// ---cut---
result.match(
  (payload) => console.log("Unsigned payload ready", payload),
  (error) => console.error("Build failed", error),
);
```
